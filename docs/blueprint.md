# **App Name**: VitalAura

## Core Features:

- User Onboarding & Profile Setup: Guides new users through a 4-step setup process to collect personal, medical, and health goal information, including profile photo upload and Google Fit connection. BMI is auto-calculated and displayed with color-coded indicators. All data is saved to Firestore.
- Personalized Smart Dashboard: Displays a 'VitalScore' based on aggregated health metrics, a grid of real-time health data cards with mini-graphs (steps, heart rate, SpO2, calories, sleep, stress, active minutes, hydration), and weekly/monthly trend charts. Features an AI-generated daily health insight tool based on user data.
- Wearable Device Sync & Manual Entry: Allows connection to Google Fit, Apple HealthKit, Fitbit, Samsung Health, or manual entry for syncing and storing health metrics (steps, heart rate, sleep, etc.) to Firestore, with auto-sync and status indicators.
- Streaks & Achievements System: Tracks daily streaks and awards collectible badges for health milestones, featuring animated notifications upon earning, with all achievements saved to Firestore.
- AI Personal Doctor (Symptom Analyzer): A chat-style interface featuring an animated AI doctor avatar. Users can input symptoms via free text or a guided structured form. An AI medical tool analyzes symptoms and patient profile to provide probable diagnoses, risk levels (LOW, MODERATE, HIGH, EMERGENCY), and structured recommendations including OTC medicines, home remedies, specialist referrals, and emergency contacts. Integrates Google Places API for nearby medical facilities. All consultations are saved to Firestore.
- AI Wellness Advisor (Ayurveda & Modern): An AI-powered wellness tool offering advice in either Ayurveda or Modern Medicine modes. Responses include personalized recommendations (herbs, yoga, diet for Ayurveda; clinically proven ingredients, supplements, lifestyle for Modern). Features a Dosha quiz for Ayurvedic personalization and allows saving responses to a personal wellness journal in Firestore.
- SOS Emergency Alert System: A critical safety feature activated via a pulsing button with a 'SLIDE TO CONFIRM' gesture. Triggers simultaneous actions: live GPS location tracking to Firebase Realtime DB, sending emergency messages via WhatsApp and Fast2SMS to contacts, displaying a prominent 'CALL 112' button, and showing nearby hospitals via Google Maps. Displays user's emergency profile and allows cancellation with a 'safe' message.
- AI Specialist Avatar Consultations: A gallery of 11 specialized AI tools, each acting as an expert consultant (e.g., Psychiatrist, Gynecologist, Cardiologist), with unique personalities and dedicated chat interfaces. Users can engage in real-time conversations, receiving structured advice and personalized guidance. Chat histories are saved to Firestore for each avatar.
- Dynamic Exercise & Workout Hub: A comprehensive workout library categorized by strength, yoga, dance, cardio, calisthenics, HIIT, and meditation, featuring embedded YouTube videos, metadata, and step-by-step guides. Includes an AI workout planner tool that generates personalized 4-week progressive plans based on user goals and preferences. Tracks workout logs, personal records, and offers community challenges, all saved to Firestore.
- Medication Manager & Reminders: Manages medication schedules, doses, and frequencies with color-coded pill cards. Provides Firebase Cloud Messaging push notifications for dose times and refill reminders. Features a medication log and an AI tool for basic drug interaction checks. All medication data and logs are saved to Firestore.
- Chandra Cycle & Reproductive Health Tracker: Logs period start/end dates and daily symptoms (flow, cramps, mood). Provides cycle predictions (next period, ovulation, fertile days) visualized on a calendar. Calculates cycle statistics and offers PMS alerts. All data is saved to Firestore and integrates with the Dr. Priya avatar.
- Anna Sutra Nutrition & Hydration Log: Tracks daily water intake with an animated bottle visualization and quick-add buttons, sending hourly reminders if needed. Features a meal logging system where an AI tool analyzes food descriptions to provide exact nutritional values (calories, protein, carbs, fat, fiber). Displays daily macro summaries and individual meal cards. All data is saved to Firestore.
- Nidan Patra Lab Report Analyzer: Allows users to upload blood test reports (PDF/image) for AI-powered analysis. An AI tool extracts parameter values, compares them against normal ranges, assigns status (Normal/High/Low), and provides plain language explanations and recommendations. Supports trend tracking for multiple reports and offers a quick action to consult an AI specialist. Reports are stored in Firebase Storage and analysis results in Firestore.
- Health Articles & Educational Content: Provides a curated section of health articles across various categories and features AI-generated daily health tips and 'Did You Know?' facts on the dashboard. Allows bookmarking articles to a reading list.
- User Profile & App Settings: Allows users to manage personal and medical information, connected devices, notification preferences, privacy settings, and language. Includes data export functionality (JSON/PDF), dark/light mode toggle, and displays subscription plan details. Supports Google Sign-In, Email, and Phone OTP authentication.

## Style Guidelines:

- Deep space-inspired animated background — deep navy to midnight purple gradient (#0a0e27 → #1a1040 → #0d1f3c) with slowly drifting, glowing blurred orbs in violet (#7C3AED), cyan (#06B6D4), and rose (#F43F5E). Primary: #7C3AED (violet/purple), Accent: #06B6D4 (electric cyan), Success: #10B981 (emerald green), Warning: #F59E0B (amber), Danger: #EF4444 (red), Surface: rgba(255,255,255,0.05) with blur, Text Primary: #F1F5F9, Text Secondary: #94A3B8. Ayurveda mode uses earthy warm tones (saffron, forest green, gold accents), Modern mode uses clean clinical whites and blues, and the SOS button is a red gradient circle (#EF4444 to #991B1B).
- The font 'Plus Jakarta Sans' from Google Fonts is used across all text. Headings are bold and large, while body text is clean and readable.
- All icons are sourced from Lucide React or Heroicons. A specific lightbulb icon is used for the AI Health Insight Card.
- The design language is Glassmorphism, featuring frosted glass cards with backdrop-filter: blur(20px), semi-transparent white/purple borders, subtle inner glows, and a soft box-shadow. Desktop layout includes a collapsible left sidebar with glowing active states and a pulsing VitalAura logo. Mobile layout features a fixed bottom navigation bar with 6 icons and an active tab that glows in violet. The application is fully responsive across phone (320px), tablet (768px), and desktop (1440px), ensuring touch targets are a minimum of 48px for mobile.
- The background features a deep space-inspired animated gradient with slowly drifting, glowing blurred orbs that float and pulse using CSS keyframe animations. All screen transitions use smooth slide/fade animations powered by Framer Motion. Loading states utilize skeleton loaders. Micro-animations include button hover lifts, card hover glows, and number count-up animations on dashboard stats. The VitalAura logo has a pulse animation, the VitalScore ring animates its fill on page load, and the streak counter includes a fire emoji animation with badge notification popups. The AI doctor avatar displays idle breathing, thinking, and nodding animations, and the SOS button is a large, pulsing, animated floating button with a continuous pulse ring.
